# 5%+ 신고 follow-trade strategy 의 backtest 발견

*이 문서는 5pct-radar 의 10년 lifecycle backtest (2016-05~2026-05, 455 CLOSED cycle) 의
**정직한** 결과 보고서입니다. 일부 결과는 5pct-radar 의 *초기 마케팅 메시지* 와
모순됩니다. 그 모순 자체를 사용자에게 정직히 보고하는 것이 본 문서의 목적입니다.*

## 🚨 가장 중요한 발견 — *행동주의 follow trade = 부정적 기대값*

### Scenario 별 IRR > 15% 적중률 (10년 표본)

| scenario (간이 분류) | N | **hit rate** | median IRR | median hold |
|---|---:|---:|---:|---:|
| 🟢 **단발진입 시즌매매** | 111 | **43.2%** | +2.0% | 153일 |
| 🟡 기타 | 111 | 34.2% | +0.5% | 461일 |
| 🟡 PE_buyout | 15 | 26.7% | −21.1% | 133일 |

**핵심**: 우리가 *"행동주의 follow trade"* 라 부른 패턴 (체계 매집 + 장기 hold) 의
hit rate 가 **7.9%** — 시장 평균 (24.5%) 의 *1/3 수준*. **인덱스 ETF 보다 못한 strategy**.

## 🎯 진정한 가치 — *단발 인수 시그널 추적*

10년 데이터에서 *IRR > 15% 적중률 높은 패턴*:

```
hold_days < 6개월 (단발 PE / 공개매수 / M&A 종결)
+ n_buys ≤ 2 (체계 매집 X, 단발 진입)
+ Q4 (10~12월) 진입
+ extract_llm 시나리오 = 산업통합M&A / PE_buyout
+ 본문 "공개매수 / 합병 / 인수계약 / 거래종결" 키워드
→ IRR > 15% hit rate 약 50~68%
```

즉 **5pct-radar 의 진짜 활용** = ***단발 인수 (M&A, PE buyout) 추적***, 행동주의 follow trade 아님.

- **평균 hold 1,500~2,000일 (4~5년)** — *single-name value 투자*
- *IRR (annualized) 환산* 시 *long hold → IRR 작아짐* (분모 시간 큼)
  - 예: 한라캐스트 raw +181.9% / 5년 → IRR **+23.5%** (raw 만 보면 *훌륭*, IRR 으로는 *적당*)
- ***Absolute return + 자금 가중* 이 진정한 평가 단위***

### 결정적 통찰 — 3가지 운용사 카테고리

| 카테고리 | 운용사 예시 | 평가 가능성 |
|---|---|---|
| **🟢 확정 양수** (closed_ratio 높음 + 양수 median) | 한국투자밸류 (97%, +4.2%) | ✅ *진짜 양수 성과* 확인 |
| **🔴 확정 음수** (closed_ratio 높음 + 음수 median) | 신영자산운용 (97%, −10.6%) | ✅ *진짜 follow 회피* 확인 |
| **🟡 판단 보류** (closed_ratio 낮음, 미실현 큼) | **VIP (47%, OPEN +54%)** | ⚠️ *cycle 완성 (~5년) 후 평가 가능*. 단기 결론 불가 |

### 함의 — 운영 결정

1. ***Closed-only IRR ranking* 은 *long-hold value 운용사* 에 unfair**
2. **VIP follow trade 결정 = *현재 portfolio 의 *미실현 winners 가 realize 될 때까지 대기**
3. **펀드 운용 차원 진짜 평가** = *gross AUM 가중 + 현재 portfolio mark-to-market* — 우리 *cycle 단위 분석* 보다 *펀드 NAV 시계열 데이터* 필요. **이는 5pct-radar 의 범위 밖**.

5. **Survivorship bias** — yfinance 가 *상장폐지 종목* 데이터 일부 누락. 진정한 base rate 보다 *낙관 편향*.

## 도구 정체성 재정의

5pct-radar 의 *초기 표기* 는 *"한국 거래소 5% 대량보유 신고 → catalyst 자동 분석"*
이었으나, 10년 backtest 결과로:

- **행동주의 follow trade = 부정적 기대값** (hit 7.9%, n=126)
- ***단발 PE / 산업통합M&A / 공개매수 신고 추적* = 양수 기대값** (hit 27~43%)

따라서 *진정한 활용 방향*:
1. *주식등의 대량보유상황보고서* 의 *시나리오 자동 분류 (extract_llm)*
2. *단발 매수 + 짧은 hold 예상* 시나리오만 *follow*
3. *체계 매집 + 장기 hold* 패턴은 *명시적 회피*
