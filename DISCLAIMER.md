# 면책 조항

## ⚠️ 투자 권유가 아닙니다

**`5pct-radar` 는 연구·교육용 소프트웨어입니다. 어떤 종목의 매수·매도 권유나
추천이 아닙니다.**

본 시스템은 공개 데이터 (DART) 와 인공지능 (Google Gemini) 으로 *5% 신고 본문의
구조화된 정보* 와 *시나리오 분류 초안* 을 만들어 줍니다. 다음에는 해당하지 않습니다:

- 금융 자문
- 증권 매매 권유
- 개인 맞춤 투자 추천
- 정확성·완전성·미래 수익에 대한 보장

## 한계와 위험

1. **인공지능 환각 가능.**
   본문 분석 (Gemini 2.5 Pro structured output) 과 시나리오 분류는 인공지능이
   수행합니다. 시스템 지시문에서 추측을 금지하지만 완전히 막을 수는 없습니다.
   **모든 보고서의 evidence 필드 (본문 인용) 를 직접 확인하세요.**

2. **공시는 *법적 사실* 만 담습니다.**
   "보유 목적 = 경영권 영향" 은 *법적 분류* 일 뿐 *진짜 의도* 가 아닙니다.
   인수자가 행동주의인지 단순 시너지 통합인지 청산인지는 *본문에 없습니다*.
   매니지먼트 인터뷰·산업 네트워크로 사람이 검증해야 합니다.

3. **12개월 EV 분포는 *시스템 초안* 입니다.**
   확률 추정은 인공지능의 *상식* 기반이고, 통계적으로 검증된 것이 아닙니다.
   "EV +16.5%" 같은 숫자는 *예측* 이 아니라 *시나리오 가중 평균의 한 추정* 입니다.

4. **그룹 구조 매핑은 1-hop 만 추적.**
   *모회사 → 자회사* 직접 관계만 정확합니다. *손자회사 → 할아버지회사* 매핑은
   놓칠 수 있습니다. 보고서 §2 에 *매핑 방법* 을 명시해 사용자 검증 가능.

5. **Google Search grounding 의 출처 검증 필요.**
   인공지능이 검색한 언론 보도가 *오보* 일 수도 있습니다. 보고서 §6 에 *출처
   URL* 을 명시해 사용자가 직접 확인할 수 있게 했습니다.

6. **상장 폐지된 종목 누락.**
   corp_code 매핑은 *현재 상장* 종목만 포함합니다. 과거 신고는 일부 매핑 실패
   가능.

7. **실제 매매 비용 미반영.**
   슬리피지, 거래세, 양도세, 시장 임팩트는 EV 분포에 포함되지 않습니다.

## 한국 주식 시장 한정

본 도구는 **한국 거래소 (유가증권시장 + 코스닥) 상장 종목 전용** 으로 설계됐습니다.
DART 공시·자본시장법을 전제로 합니다. 다른 나라 시장에 적용하려면 상당한 수정이
필요합니다.

## 사용자가 동의해야 할 것

본 소프트웨어를 사용하면 다음에 동의하는 것입니다:

- 모든 투자 결정 전 *본인이 직접* 추가 조사 수행
- 거주 국가의 모든 증권 법규 준수
- 시스템 출력의 evidence 와 source URL 을 *원본 공시* 로 재검증
- 본 소프트웨어 출력을 투자 결정의 *유일한 근거* 로 사용하지 않음
- 과거 성과가 미래 결과를 보장하지 않음을 인지
